I designed this site using Figma. The background colors are approximations of Crayola crayon colors, and I checked all color combinations using the WhoCanUse color contrast tool to make sure they are accessible to people experiencing a wide range of permanent, temporary, and situational visual disabilities.

I built this site with Jekyll to take advantage of its component-friendly templating engine. All the CSS is custom, and I kept the JavaScript to a minimum. If you want, you can check out this site's source code on GitHub.
